# Expo Router Example

Use [`expo-router`](https://expo.github.io/router) to build native navigation using files in the `app/` directory.

## 🚀 How to use

app created with:
```sh
npx create-expo-app cupid-app-test-3 -e with-router
```

to start app run:
```sh
npx expo start
```

might need to install some more packages
